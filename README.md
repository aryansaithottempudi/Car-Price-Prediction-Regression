# Car Price Prediction using Linear, Ridge, and Lasso Regression

## Overview

This project predicts car prices using three regression algorithms:

- Linear Regression
- Ridge Regression
- Lasso Regression

The objective is to compare the performance of the three models and identify the model that gives the best prediction accuracy.

## Dataset

Kaggle Dataset:
https://www.kaggle.com/datasets/hellbuoy/car-price-prediction

Dataset File:
CarPrice_Assignment.csv

## Models Used

- Linear Regression
- Ridge Regression
- Lasso Regression

## Train/Test Split

- Training Data: 80%
- Testing Data: 20%

## Evaluation Metrics

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score
